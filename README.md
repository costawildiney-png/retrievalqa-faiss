# Sistema de Recuperação de Informações com FAISS

Projeto desenvolvido utilizando LangChain e FAISS para recuperação de informações baseada em embeddings.

## Tecnologias utilizadas

- Python
- LangChain
- FAISS
- HuggingFace Embeddings

## Como executar

```bash
python -m pip install langchain langchain-community faiss-cpu sentence-transformers transformers torch
